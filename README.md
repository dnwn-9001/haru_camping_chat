# 하루캠핑
집에서 뒹구는 캠핑용품 대여하고 용돈 벌자!
<br/>
<br/>


## 프로젝트 설명
### 1. 개요
집에서 뒹굴고 있는 캠핑 용품을 대여하고 용돈 버는 플랫폼.

- 캠핑 용품 대여 게시판
- 캠핑 관련 자유 익명투표 게시판
<br/>

### 2. 사용된 기술 및 언어
- Next.js
- Redux
- Typescript
- antd
- styled-components
<br/>

## 설치 및 실행 방법

```bash
$ git clone https://github.com/dnwn-9001/haru_camping.git
$ npm install
$ npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
