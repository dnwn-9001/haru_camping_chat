# 하루캠핑
<br/>

## 프로젝트 설명
### 1. 개요
집에서 뒹굴고 있는 캠핑 용품을 대여하고 용돈 버는 플랫폼.
<br/>
<br/>

### 2. 사용된 기술
- Next.js
- Redux
- Typescript
- antd
- styled-components
<br/>

## 설치 및 실행 방법

```bash
$ git clone https://github.com/dnwn-9001/haru_camping.git
$ npm install
$ npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
